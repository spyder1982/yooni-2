Cara cepat deploy ke GitHub Pages:
1. Buat repo baru di GitHub (nama contoh: yooni-token) dan set Public.
2. Upload semua file (Add file -> Upload files), lalu Commit changes.
3. Buka Settings -> Pages -> Source: Deploy from a branch -> Branch: main -> /(root) -> Save.
4. Tunggu 1-2 menit, lalu buka link yang muncul (https://<username>.github.io/yooni-token/).

Butuh bantuan deploy? Balas "deploy" dan aku bantu langkah per langkah.
